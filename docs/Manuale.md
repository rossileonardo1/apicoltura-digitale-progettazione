Manuale di configurazione
Router STRONG 4G LTE 300 Mini
Progetto PCTO – Apicoltura Digitale
Questo manuale guida passo passo alla prima configurazione del router Strong 4G LTE 300 Mini. È pensato come aiuto pratico, con istruzioni brevi e chiare.

1. Materiale necessario
Router Strong 4G LTE 300 Mini
Alimentatore
SIM dati attiva
PC / smartphone / tablet

2. Accensione del router
Inserire la SIM dati nello slot del router
Collegare il router alla corrente elettrica
Attendere l’accensione completa (LED attivi)



3. Connessione al Wi‑Fi del router
Sul dispositivo, aprire le impostazioni Wi‑Fi
Collegarsi alla rete Wi‑Fi indicata sull’etichetta dietro al router
Inserire la password Wi‑Fi di default (scritta dietro al router)




4. Accesso al pannello di configurazione
Aprire un browser (Chrome, Edge, Firefox…)
Nella barra degli indirizzi digitare:
IP del router (es. 192.168.0.1)
(l’indirizzo è indicato dietro al router)
Premere Invio


5. Login al router
Si apre la pagina di login Strong
Cliccare su Login
Inserire username e password di default (sull’etichetta del router)



6. Cambio nome rete Wi‑Fi (SSID)
Entrare in Impostazioni rapide
Modificare il campo SSID (nome della rete)
Cliccare Avanti / Next


7. Cambio password Wi‑Fi
Nella schermata successiva modificare la Chiave di sicurezza Wi‑Fi
Inserire una password sicura
Cliccare Applica

8. Cambio credenziali di accesso al router (consigliato)
Andare in Impostazioni di sistema
Cambiare:
Nome utente
Password di accesso
Compilare i campi richiesti
Cliccare Applica
Questa operazione aumenta la sicurezza del router




9. Configurazione DHCP (max 4 dispositivi)
Aprire Impostazioni dispositivo
Menu sinistro → Impostazioni avanzate
Modificare l’IP Pool DHCP
Esempio per 4 dispositivi:
IP iniziale: 192.168.0.100
IP finale: 192.168.0.103
Salvare le impostazioni

10. Verifica banda Wi‑Fi (2.4 GHz)
Il router utilizza banda 2.4 GHz (massima supportata)
Per visualizzarla:
Tornare alla pagina principale
Sezione Wi‑Fi (a destra)
Cliccare Impostazioni
Menu sinistro → Impostazioni avanzate




11. Impostazioni SSID avanzate
In alternativa:
Menu SSID Settings
Modificare:
Nome rete (SSID)
Password Wi‑Fi

12. Conclusione
Il router Strong 4G LTE 300 Mini è ora:
Configurato
Sicuro
Pronto per l’uso in un progetto di apicoltura digitale

Le schermate mostrate nelle immagini aiutano a riconoscere le varie sezioni del menu.
